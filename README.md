E-Tutorial 
A Java-based web application built using JSP, Servlets, and MySQL, designed to simplify the management of courses, users, and events for educational institutions.
This system provides a clean and efficient way to handle student information, course details, and admin operations all in one place.

Features
User Authentication – Secure login for students and admins
Course Management – Add, view, and update course details
Events & Announcements – Display upcoming events and updates
Contact & Feedback – Interactive pages for communication
Modular Structure – Organized using MVC architecture

Tech Stack
Frontend: JSP, HTML, CSS, JavaScript
Backend: Java Servlets
Database: MySQL
Server: Apache Tomcat
